PROTÓTIPO AMANO - WEB APP LEVE

Formato escolhido:
- HTML + CSS + JavaScript puro, em um único arquivo index.html.
- Sem React, sem Flutter e sem biblioteca externa.
- Mais leve para abrir no celular e simples para publicar no Firebase Hosting.

Como testar:
1. Abra o arquivo index.html no navegador.
2. Na tela Cliente, adicione pratos ao carrinho.
3. Clique em "Ver pedido".
4. Preencha nome, WhatsApp, endereço e pagamento.
5. Finalize o pedido.
6. Clique em "Ver na cozinha" ou mude para a aba Cozinha no topo.

Importante:
- Esta versão salva os pedidos no localStorage do navegador apenas para protótipo.
- Para produção, o localStorage deve ser substituído por Firebase Firestore.
- Não tem login de cliente nesta versão, conforme solicitado.
- O painel da cozinha está no mesmo arquivo para facilitar o teste.
